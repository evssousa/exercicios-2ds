# 📝 Atividade rápida

## Site com Box Model

Utilizando a mesma estrutura:

```
index.html
/src/css/style.css
```

---

## 🔹 index.html deve conter:

- Estrutura HTML completa
- Link para `style.css`
- Um `<div>` com o texto:

```
Aprendendo Box Model
```

---

## 🔹 style.css deve conter no seletor `div`:

- `background-color`
- `border`
- `padding`
- `margin`

Todos com valores definidos.
