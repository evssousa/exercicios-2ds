# 📝 Atividade rápida

## Criando site com CSS

Crie o projeto com a seguinte estrutura:

```
index.html
/src/css/style.css
```

---

## index.html deve conter:

- Estrutura HTML completa
- `<h1>` com o texto:

```
Minha Primeira Página com CSS
```

- Um `<p>` com qualquer texto
- Link correto para `./src/css/style.css`

---

## style.css deve conter:

- `body` com `background-color`
- `h1` com:
    - `color`
    - `border`
- `p` com `color`
