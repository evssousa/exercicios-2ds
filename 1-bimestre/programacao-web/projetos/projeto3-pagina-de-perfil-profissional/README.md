# 📘 PROJETO 3: Página de Perfil Profissional

# 🧩 PROBLEMA

Uma empresa chamada **DevConnect** deseja criar uma página simples de **perfil profissional** para desenvolvedores iniciantes.

Essa página será utilizada internamente para apresentar novos alunos da empresa.

Seu trabalho é desenvolver uma **página de perfil estilizada usando HTML e CSS**.

# 📋 REQUISITOS DO PROJETO

Crie uma página contendo as seguintes seções.

# 🔹 1. Título da Página

Deve existir um título com o texto “Perfil do Desenvolvedor”, tendo que possuir uma classe “titulo”.

# 🔹 2. Seção de Informações

Criar uma `<div>` com classe “perfil”. Dentro dela devem existir:

- Um subtítulo com “Nome do Desenvolvedor”
- Um parágrafo com “Desenvolvedor Web”, tendo uma classe “profissao”
- Outro parágrafo com “Apaixonado por tecnologia e programação”, tendo uma classe “descricao”

# 🔹 3. Seção de Habilidades

Criar uma lista **não ordenada** contendo pelo menos 3 habilidades.

Exemplo (você poderá modificar os valores):

```
HTML
CSS
JavaScript
```

# 🎨 REQUISITOS DE CSS

## 🔹 Estilo do body

Deve possuir uma cor de fundo “Thistle” e fonte “Verdana”.

## 🔹 Estilo do título

O título deve ter o alinhamento para esquerda, cor preta e margem de 5px em todos os lados.

## 🔹 Estilo da div perfil

A div perfil deve conter uma borda de 2px, um espaçamento de 10px e margem de 5px, com a cor de fundo “Gainsboro”.

## 🔹 Estilo da profissão

O parágrafo pofissao receberá o tamanho da fonte 12px e a cor do texto preta.

## 🔹 Estilo do botão

E a âncora do botão terá um `display: inline-block`, com espaçamento de 12px, cor de fundo “Tan”, cor do texto preta e uma borda de 2px.
