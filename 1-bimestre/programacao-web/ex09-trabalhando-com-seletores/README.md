# 📝 Atividade Rápida

## Trabalhando com Seletores

Crie o projeto com a estrutura:

```
index.html
/src/css/style.css
```

---

## 📄 index.html deve conter

- Um `<h1>` com id **titulo**
- Um `<p>` com classe **destaque**
- Um `<p>` normal

Exemplo:

```html
<h1 id="titulo">Página de Estilos</h1>

<p class="destaque">Texto importante</p>

<p>Texto comum</p>
```

---

## 📄 style.css deve conter

### Estilo para `#titulo`

- `text-align`
- `font-size`

---

### Estilo para `.destaque`

- `color`
- `font-weight`

---

### Estilo para `p`

- `font-family`
