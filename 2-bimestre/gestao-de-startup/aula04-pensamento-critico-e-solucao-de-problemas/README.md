# 📘 AULA 24: PENSAMENTO CRÍTICO E SOLUÇÃO DE PROBLEMAS

## 🎯 Objetivos da aula

- Entender o que é pensamento crítico
- Aprender a analisar problemas com lógica
- Desenvolver tomada de decisão consciente
- Aplicar soluções práticas em situações reais

## 🧠 O que é pensamento crítico?

Pensamento crítico é:

> Analisar situações antes de aceitar respostas prontas.
> 

É questionar:

- Isso faz sentido?
- Qual é a causa real?
- Existe outra solução melhor?

## 📌 No mercado de trabalho isso é valioso porque:

Empresas querem profissionais que:

- pensem antes de agir
- resolvam problemas
- tomem decisões inteligentes
- não dependam de ordens para tudo

## 🔍 Como analisar um problema

Use este passo a passo:

```
1. Qual é o problema?
2. O que causou isso?
3. Quais opções existem?
4. Qual é a melhor solução?
5. Como evitar no futuro?
```

## 📌 Exemplo prático

Problema:

> O sistema da empresa está lento.
> 

Pensamento comum:

> “Precisa trocar tudo.”
> 

Pensamento crítico:

- Está lento em qual parte?
- Desde quando?
- Todos os usuários reclamam?
- O problema é internet? servidor? código?

👉 Só depois disso se decide.

## ⚠️ Erros comuns sem pensamento crítico

- agir por impulso
- culpar pessoas sem investigar
- aceitar primeira solução
- repetir erros antigos

## 🧠 Solução de problemas em startups

Startups valorizam quem:

```
✔ identifica problemas
✔ propõe soluções
✔ testa alternativas
✔ aprende com erros
```
