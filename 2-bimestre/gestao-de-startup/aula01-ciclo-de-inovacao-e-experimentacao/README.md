# 📘 AULA 21: CICLO DE INOVAÇÃO E EXPERIMENTAÇÃO

## 🎯 Objetivos da aula

- Entender como funciona o **ciclo de inovação**
- Compreender a importância da **experimentação em startups**
- Identificar como ideias são testadas e melhoradas
- Desenvolver mentalidade de melhoria contínua

## 💡 O que é inovação?

Inovação é:

> Criar ou melhorar algo que gere valor para as pessoas.
> 

Não precisa ser algo revolucionário.

📌 Pode ser:

- melhorar um sistema
- facilitar uma tarefa
- reduzir tempo de execução

## 🧪 O que é experimentação?

Experimentação é:

> Testar ideias na prática para ver se funcionam.
> 

Em startups, não se espera perfeição.

👉 A lógica é:

- testar rápido
- aprender rápido
- melhorar rápido

## 🔄 O ciclo de inovação

O processo segue um ciclo contínuo:

```
IDEIA → TESTE → FEEDBACK → MELHORIA → NOVO TESTE
```

### Explicando cada etapa:

- **Ideia** → solução para um problema
- **Teste** → colocar a ideia em prática (mesmo simples)
- **Feedback** → ouvir usuários
- **Melhoria** → ajustar a solução
- **Novo teste** → repetir o ciclo

📌 Esse ciclo pode acontecer várias vezes.

## 📌 Exemplo prático

Imagine um app de estudos:

1. Criam uma funcionalidade simples
2. Alunos testam
3. Recebem opiniões
4. Ajustam o app

👉 Esse processo se repete até melhorar.

## ⚠️ Erro comum

Esperar o produto ficar perfeito antes de mostrar.

❌ Errado:

> “Vamos terminar tudo primeiro.”
> 

✅ Correto:

> “Vamos testar uma versão simples primeiro.”
> 

## 🧠 Mentalidade correta

```
Testar → Aprender → Melhorar → Repetir
```
